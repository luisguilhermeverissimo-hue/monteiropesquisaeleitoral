Configure a planilha

Abra pesqele-dashboard.html em um editor de texto e localize a linha:

const URL_DADOS = "https://docs.google.com/spreadsheets/d/1dutvbKFOE2kcvyUmcwraBGMy0fvZHlw2Sz-vBFDDQF4/gviz/tq?tqx=out:csv&gid=1041307058";

Substitua pela URL gerada em Arquivo → Compartilhar → Publicar na Web → aba "dados" → CSV → Publicar.

Execute atualizar_pesqele.py ao menos uma vez para popular a planilha antes de abrir o dashboard.

Ou carregue dados manualmente enquanto configura:

  1. Acesse dadosabertos.tse.jus.br
  2. Baixe pesquisa_eleitoral_2026.zip e extraia
  3. Clique em 📂 Manual na barra acima
